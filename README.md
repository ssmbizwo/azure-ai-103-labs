# Azure AI Engineering — AI-103 Labs

A hands-on lab notebook for [Microsoft AI-103: Developing AI Apps and Agents on Azure](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/ai-103).

Each folder is a working lab tied to a specific exam objective, with a README that
explains what was built, what evaluation showed, and what would break it.

## Stack
Python · Azure AI Foundry · Azure OpenAI · Azure AI Search · Azure AI Agents · `azure-ai-evaluation` · OpenTelemetry · Docker · GitHub Actions

## Quickstart
1. `cp .env.example .env` and fill in your own Foundry credentials.
2. `python -m venv .venv && source .venv/bin/activate`
3. `pip install -r requirements.txt` *(coming in Week 1)*
4. `cd week-N-…` and follow its README.
